= A Simple Caesar Cipher =

This application provides a functional Caesar Cipher that can be used to encrypt 
and decrypt text, using a variable shift size.

* Running the application:

The application can be run using the command "java -jar dist/CaesarCipher.jar"

* Instructions

The application consists of two tabs: one for encryption and one for decryption.
On each tab are located buttons for encrypting/decryption the provided text,
as well as clearing the text in that tab. The shift size for the cipher can
be specified using the "Cipher Shift" control, located at the top. To close
the applicaiton, either exit the window use click the "Exit" item in the "File"
menu.
